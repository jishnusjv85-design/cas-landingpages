# Complete Aqua Solutions - Landing Page

A modern, responsive landing page for Complete Aqua Solutions water purifier business in Calicut, Pantheerankave.

## Features

- **Modern Design**: Clean, professional design with smooth animations
- **Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Product Showcase**: Display area for 9 premium water purifier products
- **Contact Integration**: Direct phone call and WhatsApp integration
- **Smooth Scrolling**: Enhanced user experience with smooth navigation
- **SEO Friendly**: Proper HTML structure and meta tags

## Setup Instructions

1. **Add Product Images**: 
   - Create an `images` folder in the root directory
   - Add your product images with the following names:
     - `product-1.jpg` (Ever Pure)
     - `product-2.jpg` (Whitney Wave)
     - `product-3.jpg` (Pure Arroyo)
     - `product-4.jpg` (Copper Pure Wave)
     - `product-5.jpg` (Aqua Purito)
     - `product-6.jpg` (Orwell Misty)
     - `product-7.jpg` (Aqua Doris)
     - `product-8.jpg` (Copper Pearl)
     - `product-9.jpg` (CAS Viva)

2. **Open the Website**:
   - Simply open `index.html` in a web browser
   - Or use a local server for better performance

## File Structure

```
Landing-pages/
├── index.html          # Main HTML file
├── styles.css          # All styling
├── script.js           # JavaScript functionality
├── README.md           # This file
└── images/             # Product images folder (create this)
    ├── product-1.jpg
    ├── product-2.jpg
    └── ...
```

## Customization

- **Colors**: Edit CSS variables in `styles.css` (lines 4-11)
- **Content**: Modify text content in `index.html`
- **Contact Info**: Update phone number and location in `index.html` and `script.js`
