## 🚀 Enable GitHub Pages (Get Your Clickable URL)

### Step 1: Go to Repository Settings
1. Open: **https://github.com/jishnusjv85-design/cas-landingpages**
2. Click **"Settings"** tab (top right)
3. Scroll down to **"Pages"** section (left sidebar)

### Step 2: Configure GitHub Pages
1. Under **"Source"**, select **"Deploy from a branch"**
2. Select **"main"** branch
3. Select **"/ (root)"** folder
4. Click **"Save"**

### Step 3: Wait for Deployment
- Wait 2-5 minutes for GitHub to build your site
- Refresh the Pages settings page
- You'll see: **"Your site is live at..."**

### Step 4: Get Your URL
Your landing page will be available at:
**https://jishnusjv85-design.github.io/cas-landingpages/**
